OpenClaw 版本更新笔记（Unreleased → 2026.4.x）

来源: CHANGELOG.md | 更新日期: 2026-04-05

Unreleased 重大特性

🧠 Memory/Dreaming（实验性）

  • 加权短期回忆提升（short-term recall promotion）
  • 托管梦境模式：off|core|rem|deep
  • /dreaming 命令 + Dreams UI
  • 多语言概念标签

🗣️ MiniMax TTS

  • 通过 T2A v2 API 支持 MiniMax 原生语音合成
  • bundled MiniMax speech provider

🤖 Claude CLI 增强

  • 通过 loopback MCP bridge 开放 OpenClaw 工具给 Claude CLI
  • stdin + stream-json 流式输出（告别 argv 传递）
  • 非交互式 --permission-mode bypassPermissions 修复（cron/heartbeat 使用）

⚡ Prompt Cache 优化

  • 跨 transport fallback 复用前缀
  • 确定性 MCP tool 顺序
  • 压缩前后 cache 复用更稳定

🔌 新 Provider

  • StepFun（中国/全球端点，step-3.5-flash）
  • Ollama Web Search（无需 key）
  • MiniMax Search（Coding Plan search API）

🛠️ 重要 Bug 修复

  • Cron：断续 recurring jobs 在第一次网关重启时即重放
  • 子智能体：exec-approvals.json 权限修复（security: "full"）
  • 安全：plugin-only allowlists 修复，关闭 SSRF redirect bypass
  • Telegram：current-model 检查修复，HTML 格式修复
  • Discord：proxy 维持，webhook media 发送修复

2026.4.2 重大变更

Breaking Changes

  • xAI: x_search 设置从 tools.web.x_search.* 迁移到 plugins.entries.xai.config.xSearch.*
  • Firecrawl: web_fetch 从 tools.web.fetch.firecrawl.* 迁移到 plugins.entries.firecrawl.config.webFetch.*
  • 迁移命令：openclaw doctor --fix

新功能

  • Task Flow 基础设施恢复（托管任务流程 + 子任务派生）
  • Android Google Assistant App Actions 支持
  • Exec 默认模式改为 YOLO（security=full, ask=off）
  • before_agent_reply hook（插件可短路 LLM 回复）

版本时间线

版本 日期 关键特性

2026.4.2 2026-04 Task Flow, Android assistant, xAI/Firecrawl breaking change
