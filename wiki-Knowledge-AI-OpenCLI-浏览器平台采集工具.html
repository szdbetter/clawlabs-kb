OpenCLI-浏览器平台采集工具

← 返回首页

OpenCLI · AI 的"手"——浏览器平台采集 CLI

stage/evergreen #domain/ai #domain/tech

来源：微信公众号《Claude 最缺的东西被这个 CLI 工具补上了》 原文：https://mp.weixin.qq.com/s/HBynzYu96bCevG94Va2fzQ 入库时间：2026-04-07

核心定位

Claude-Code 和 AI Agent 能"说"但不能"做"，OpenCLI 补上了这段执行力： - 复用 Chrome 浏览器的已有登录态，不需要重新鉴权 - 430 个内置命令，覆盖 70+ 平台（B站、知乎、小红书、Twitter/X、微信、YouTube 等） - GitHub 12.5k ⭐，零 API 费用

核心优势 vs Playwright MCP

OpenCLI Playwright MCP
执行方式 预封装命令，直接取数 AI 实时推理页面操作
速度 快 慢
Token 消耗 极低 高
稳定性 高（命令化） 低（页面变动即失效）
适合场景 工作流/定时任务 一次性复杂操作

对 ClawLabs 的价值评估

✅ 高价值点

  1. 微信公众号文章下载（最直接） bash opencli weixin download --url "文章链接" --output ./weixin # 图片一并下载，直接转 Markdown 入库 → 解决当前 Tavily 抓微信有时被验证拦截的问题

  2. 补充 Research 系统缺失平台

  3. 知乎热榜：opencli zhihu hot --limit 20
  4. B站热榜：opencli bilibili hot --limit 10

  5. 即刻：支持 → 当前 ClawLabs 情报站覆盖 小红书/Reddit/Twitter，缺知乎/B站/即刻

  6. YouTube 字幕提取 bash opencli youtube transcript --url "视频链接" → 比 agent-reach 的 YouTube 更直接

⚠️ 限制

  • 必须 Chrome 保持开启 + 已登录目标平台
  • VPS 上不能直接跑（需要桌面 Chrome 环境）
  • 需要安装 Chrome 扩展（opencli-extension.zip）
  • 写操作（发推/发笔记）需谨慎确认

安装方式（Mac/本地）

# 1. Node.js 20+
node -v

# 2. 安装
npm install -g @jackwener/opencli

# 3. 安装 Chrome 扩展（GitHub Releases 下载 zip，chrome://extensions/ 加载）

# 4. 验证
opencli doctor

常用命令速查

# 热榜
opencli bilibili hot --limit 10
opencli zhihu hot --limit 20
opencli hackernews top --limit 10

# 搜索
opencli xiaohongshu search query="AI工具"
opencli bilibili search --keyword "Claude Code" --limit 10
opencli twitter search "keyword" --limit 15 -f json

# 下载存档
opencli weixin download --url "文章链接" -o ./weixin     # 微信（含图片）
opencli zhihu download --url "文章链接" -o ./zhihu
opencli xiaohongshu download --url "笔记链接"

# 私人数据
opencli bilibili history --limit 20 -f json
opencli twitter bookmarks --limit 20 -f json

与 ClawLabs 现有工具对比

平台 现有方案 OpenCLI 方案 建议
Twitter/X xreach（cookie） opencli twitter 两个都行，xreach 更成熟
小红书 XHS MCP（18060） opencli xiaohongshu MCP 在 VPS，更稳定
B站 agent-reach（Exa付费） opencli bilibili OpenCLI 免费，优先用
知乎 ❌无 opencli zhihu 新增能力
微信公众号 Tavily（有时失败） opencli weixin 补充方案
YouTube agent-reach（Exa付费） opencli youtube OpenCLI 免费，优先用
抖音/TikTok ❌无 ❌OpenCLI也无 仍是空白

接入 ClawLabs 的建议方案

由于 OpenCLI 需要桌面 Chrome，建议在 Mac（本地）安装，通过以下方式与 VPS 协作： 1. 本地定时跑 opencli bilibili hot && opencli zhihu hot，结果推送到 VPS 2. 或在 clawlabs_dashboard 里加一个 /api/opencli 端点，调用远端 Mac 服务
