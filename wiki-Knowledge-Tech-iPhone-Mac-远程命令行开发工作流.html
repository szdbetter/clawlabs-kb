iPhone + Mac 远程命令行开发工作流

stage/seedling #domain/tech #type/summary

来源：微信公众号 入库时间：2026-04-08

核心洞察

只要三个工具（Termius + Tailscale + tmux），就能让 iPhone 变成随时随地的命令行开发终端。关键不是 SSH 本身，而是 tmux 让断线=暂停而非丢失。

移动开发不需要 iPad 或笔记本。对于 CLI 重度用户，iPhone 通过 SSH 连接远程主机，配合 tmux 会话保持，就能实现完整的命令行开发体验。三件套各司其职：Tailscale 解决网络可达性（无需公网 IP、无需端口映射），Termius 提供手机端 SSH 客户端，tmux 保证工作状态持久化。

关键要点

  1. 三件套分工明确：Tailscale（组网通路）→ Termius（SSH 客户端）→ tmux（会话保持）
  2. tmux 是核心：断线、切 App、锁屏都不丢状态，回来 tmux attach -t main 即可恢复
  3. Termius 免费版够用：不需要付费功能，核心 SSH 连接完全免费
  4. 适用场景明确：适合 CLI 重度用户（vim/neovim 编辑、git 操作、服务器管理），不适合需要 GUI 的场景

实操指南

前置准备

  • 远程主机：Mac mini / Linux VPS / 任何有 SSH 的机器
  • 手机：安装 Termius
  • 两端：都安装 Tailscale 并登录同一账户

日常工作流

# 1. 手机打开 Termius，连接远程主机（通过 Tailscale IP）
ssh user@100.x.x.x

# 2. 恢复之前的工作会话
tmux attach -t main
# 如果没有会话，创建一个
tmux new -s main

# 3. 正常工作（编辑代码、跑命令、看日志）

# 4. 离开时优雅分离（不是关闭！）
# Ctrl-b d

# 5. 下次回来，再 attach，一切还在

tmux 常用操作

操作 快捷键
分离会话 Ctrl-b d
新建窗口 Ctrl-b c
切换窗口 Ctrl-b 数字
水平分屏 Ctrl-b "
垂直分屏 Ctrl-b %
切换面板 Ctrl-b 方向键

应用场景

  • 通勤路上：手机 SSH 进服务器处理紧急问题
  • 外出时：不带电脑也能完成 CLI 工作
  • 多设备切换：从手机切到电脑，tmux 会话无缝衔接

个人评论

这套方案的核心价值不在于"手机能写代码"，而在于"永远不断线"。tmux 把远程会话从"连接依赖"变成了"状态持久"，网络只是传输层。对于我们的 VPS（已有 Tailscale + tmux），这套方案天然可用，手机端只需装个 Termius 即可。

相关概念

  • Tailscale — 零配置组网，解决内网穿透
  • tmux — 终端复用器，会话持久化的核心
  • Termius — 移动端 SSH 客户端
  • Claude-Code — CLI 开发工具，完美适配此工作流
