ClawLabs Research 情报站系统架构

创建日期：2026-03-25 来源：dashboard-mvp 项目优化计划 [来源: Claude-Code / workspace/dashboard-mvp]

一、系统定位

ClawLabs 情报站是一个AI+Crypto热点追踪平台，聚合小红书/X(Twitter)/Reddit三平台内容，通过AI分析提炼每日热点信号、爆款选题与投资机会。

访问路径：通过 Cloudflare Tunnel + nginx 提供公网访问，auth-gate 密码保护。

二、技术架构

openclaw agent (定时采集，每日22:00 BJT)
  ↓ 
scripts/collect-research.js (采集小红书/Reddit/Twitter)
  ↓
data/research/ (持久化JSON文件，每天1个)
  ↓
scripts/enrich-research.py (qwen3:8b AI分析补全)
  → signals（三大共振信号）
  → daily_review（信号评分）
  → top_opportunity（今日机会）
  → Telegram 通知推送（阈值触发）
  ↓
dashboard-mvp Express Server (port 1980)
  → /api/research/latest
  → /api/research/list  
  → /api/research/collect
  ↓
public/research/index.html (前端动态加载)
  ↓
nginx (:80) → Cloudflare Tunnel

三、数据 Schema

{
  "date": "YYYY-MM-DD",
  "time": "HH:MM BJ",
  "stats": { "total": N, "xhs": N, "reddit": N, "twitter": N },
  "xhs": [{ "platform", "category", "title", "score", "likes", "url", "rank" }],
  "reddit": [...],
  "twitter": [...],
  "signals": [{ "icon", "tag", "color", "title", "desc" }],
  "analysis": "markdown string",
  "daily_review": [{ "signal", "score", "comment" }],
  "top_opportunity": { "title", "urgency", "subtitle", "actions": [] }
}

五、关键技术决策

决策 选择 理由
数据存储 JSON 文件（不用 DB） 每天1个文件，数量少，简单可靠
AI分析模型 qwen3:8b via Ollama 免费，53 t/s，中文质量好
前端框架 纯 JS（单文件） 不引入框架，减少复杂度
