Polymarket Alpha轮循策略

来源：Discord #Polymarket Alpha轮循 (thread_id: 1480959355531559004, 307条) 提取时间：2026-03-20

一、策略核心逻辑

Alpha来源

不追热度，找规则漏洞： - 高成交量（>$10M）、未结束、3月内结束的盘口 - 规则定义存在明显争议（时间、定义、来源、边界四类模糊性） - 市场存在错误定价（把叙事直接打成价格）

四类规则模糊性

  1. 时间模糊性："年底前"、"选举日后"等未精确定义
  2. 定义模糊性："重大"、"官方认定"、"首次"等主观词汇
  3. 来源模糊性：结算依赖的"官方来源"未明确
  4. 边界模糊性：事件发生的边界条件未覆盖

二、Polymarket Skill

安装与使用

  • 来源：https://clawhub.ai/deanpress/polymarket-odds
  • 调用方式： bash node skills/polymarket/polymarket.mjs search "Trump" node skills/polymarket/polymarket.mjs events --tag=politics --limit=10 node skills/polymarket/polymarket.mjs market <market-slug> node skills/polymarket/polymarket.mjs price <token_id> node skills/polymarket/polymarket.mjs book <token_id>
  • 适用：赔率/事件查询，不替代规则模糊性分析和Alpha扫描

API数据来源

  • gamma-api.polymarket.com（市场/事件）
  • clob.polymarket.com（价格/订单簿）

三、3月Alpha候选盘口（2026-03-11筛选）

高优先级

Russia x Ukraine ceasefire by March 31, 2026? - 成交量：$24.1M - 当前价格：Yes 2.1% / No 97.9% - 争议点：停火定义（正式签署/实际停火/局部/临时/口头宣布/第三方确认） - Alpha逻辑：市场可能将"几乎不可能"直接压成2%，但真实结算路径未必那么窄

MicroStrategy sells any Bitcoin by March 31, 2026? - 成交量：相关总量$21.1M - 当前价格：Yes 1.1% / No 99.0% - 争议点："sells"定义（托管层操作/技术性转移/抵押清算/被动处置，以链上/公告/财报哪种口径为准） - Alpha逻辑：市场把"叙事上不会卖"直接打成"规则上不可能卖"

中优先级

Kraken IPO by March 31, 2026? - 成交量：$1.11M（偏小） - IPO定义争议：递表/定价/挂牌首日/直接上市/SPAC算不算

四、3月轮循执行架构

工作流

  1. Stage 1：高交易量市场筛选（交易量>$1M的活跃市场，取前20-30个）
  2. Stage 2：规则模糊性语义分析（并行子智能体，每个分析8-10个市场）
  3. Stage 3：收益机会评估（潜在收益>$10K的机会）

评估维度

  • 模糊性争议度（社区讨论热度）
  • 市场规模（交易量越大收益空间越大）
  • 当前价格偏离度（是否已反映争议）
  • 时间窗口（越临近结算，模糊性价值越高）

分工原则

  • Fiona节点：物理执行节点（高频扫描、抓取、运行脚本）
  • Reese（上层智能体）：规则争议判断、错价识别、优先级决策
  • 本地Dashboard：127.0.0.1:1980，可视化进度跟踪
