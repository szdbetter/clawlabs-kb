2026-03-17 全日复盘

这一天比看起来要密。7个战场同时推进。

🧩 业务全景

1. 闲鱼电影票项目 — Chrome CDP 攻坚

背景：要自动化接单，需要用CDP协议控制Fiona上用户已登录的Chrome。 今日进展： - 发现核心矛盾：OpenClaw browser proxy 占用 9222 端口，导致 debug port 冲突 - 解决思路：让 Chrome 改用 9300 端口 → 在 profiles.user 配置 cdpUrl 指向 9300 - 成功解密闲鱼 Cookie（tracknick=dbetter 已确认），技术路径通了 - 阻塞点：Fiona 是 Mac，GUI 应用（Chrome）必须用户手动启动，VPS 无法远程拉起 - 待做：你启动 Chrome 加 --remote-debugging-port=9300，整个链路就打通

2. 美伊盘口套利监控系统 — 上线

背景：Polymarket 上跑美伊冲突相关盘口，存在跨市场套利机会。 今日完成： - 后端 4个API路由全部上线（获取盘口、套利分析、规则解读、Manifold参考价） - 逻辑约束检验上线（4组一致性规则，今日全部满足） - 前端套利监控页面上线，接入主站侧边栏 - 真实信号：美军进入伊朗，Manifold 60-74% vs Polymarket 24.5%，价差 +35-50% - 价值：这是实打实的alpha，Manifold 为方向参考，Polymarket 才能下注

3. 个人知识库体系建设 — 架构确立

背景：每天的思考太散，需要一个能自动沉淀的知识库。 今日完成： - 选型：Obsidian 作为底座（Markdown + 双向链接） - 三层架构确立：OpenViking 语义记忆 → VPS Workspace → Obsidian Vault - 路径标准化完成：统一迁移至 /root/.openclaw/obsidian-vault/ - 写入 OpenViking 长期记忆，全channel共享 - 今天的意义：把"我说话，AI帮我整理"的闭环跑通了第一遍

4. KlingAI 电商视频生成 — 两批次交付

背景：电商频道需要产品虚拟试穿视频。 今日完成： - 第一批：3支5秒视频（esseactive系列）已发出 - 第二批：3支10秒视频（运动服装）→ 压缩为 webm 后发出（Discord 8MB 限制） - 技术坑：api-beijing.klingai.com 才是有效端点，原模特图需压缩到~100KB - 遗留：虚拟试衣功能需单独购买额度，与视频额度独立

5. VPS Claude Max 专属代理 — 配置完成

背景：需要 Claude 流量统一走日本 VPS IP 出去，避免账号风险。 今日完成： - 3x-ui v2.8.11 + VLESS-Reality 443端口配置完成 - Clash Verge 分流脚本提供（claude.ai / anthropic.com 走代理） - 待用户完成：腾讯云安全组放行443入站，Clash 切「规则」模式

6. LLM 路由分发研究 — 费曼理解落地

背景：研究 Plano 智能路由中间件，评估是否值得集成。 今日收获： - 核心理解：Plano = 医院分诊台，内置 1.5B 小模型做意图分类，不是生成模型 - 价值：简单任务省 ~80% API 成本，复杂任务质量不降 - 费曼法验证有效：类比解释秒懂，已写入规范作为常用输出方法 - 结论：现有体系已有类似分工逻辑，需做一次费用审计再决定是否上路由层

7. 工作体系完善 — 「毛选skill」建立

背景：看到一个毛泽东任务拆解范式，评估是否值得收录。 今日完成： - 提取5条铁律核心（任务明晰/时间卡死/压活心疼/主动给资源/放权兜底） - 与现有体系对比：精华已内嵌，部分原则对AI多余 - 新增skill：skills/mao-task/ 已建立 - 触发方式：说「调用毛选skill：XXX任务」，输出完整5铁律拆解

💡 今日三条思考

思考一：工具不是越多越好，边界最重要 今天在 Chrome CDP / KlingAI / Plano 上踩了不少坑，根源都是同一个问题：边界没想清楚。哪个工具负责什么、数据怎么流转，必须前置定义，否则每次都在解决同一类问题。

思考二：知识库是生长出来的，不是整理出来的 今天把 Obsidian 体系跑通，最深的感受是：捕获的门槛要极低。说几句话，AI帮整理——这才是能持续的模式。传统的"周末找时间整理"根本不可能坚持。

思考三：套利信号需要多源验证 美伊盘口 Manifold vs Polymarket 价差 35-50%，数字很诱人。但 Manifold 是虚拟货币市场，流动性极低，只能做方向参考，不能直接套利。任何信号，先问数据源的质量。

  • [ ] 腾讯云安全组 443 入站确认生效（Claude 代理最后一步）
  • [ ] KlingAI 虚拟试衣额度购买决策
  • [ ] Plano 路由集成评估：先做费用审计，看简单任务占比
