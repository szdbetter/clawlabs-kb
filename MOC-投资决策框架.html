一、五大师框架：各委员的核心视角与应用场景

委员会成员定位

委员 核心视角 最强场景 杀手问题
芒格 生意本质/护城河 判断企业质地，辨别竞争优势的可持续性 "护城河是什么？10年后还在吗？"
马克斯 风险/周期/情绪 判断市场位置，钟摆理论，第二层次思维 "当前周期在哪？大家都知道这件事了吗？"
段永平 本分/纪律/十年持有 过滤投机冲动，确认真正理解的生意 "10年后会不会更好？我愿意持有10年吗？"
德鲁肯米勒 时机/催化剂/仓位管理 确定入场时机，设置止损，管理仓位 "最关键的催化剂是什么？止损线在哪？"
西蒙斯 量化/统计/胜率 验证技术面支持，计算统计胜率 "历史形态胜率多少？期望值为正吗？"

实战应用案例

BABA（2026-03-18，$136.19） → 西蒙斯动量看空 → 触发软否决 → 分批建仓（不一次重仓） → 五委员综合评分6.0/10，等待财报后$120-125建仓25-30% → 详见 美股研究笔记

贵州茅台（¥1468.8） → 芒格+段永平高度认同（各8/10），西蒙斯放量收跌信号偏弱（6/10） → 综合7.1/10，⭐⭐⭐⭐可建仓，止损¥1385 → 详见 投资综合讨论精华

EWY韩国ETF（$131.88） → 五委员罕见高度一致（均4-4.5/10） → 回避/等待，韩元贬值+放量破MA20双重信号 → 详见 美股研究笔记

$CARDS（Collector Crypt，Solana） → 综合评分4.5/10，FDV/MC=7.8x红旗 → 有潜在价值但100%一致"当前观望" → 详见 Cards-CryptoCollect项目

二、信息优势三层理论

来自 Polymarket套利策略研究 的核心框架：

层级 1 ❌ 数据速度差（低价值）
├── NOAA等公开数据，已被bot覆盖
├── BTC 5分钟快市：10% taker费，2-5%信息优势根本覆盖不了
└── 天气市场已过度竞争，红海形成

层级 2 ⚠️ 数据组合差（中价值）
├── 3-5个数据源组合算概率
├── 少数人做，但护城河不深
└── 案例：EIA原油库存+季节性+期货曲线组合

层级 3 ✅ 规则理解差（高价值）
├── 解读盘口resolution criteria，90%人不认真读
├── "年底前"、"官方认定"、"重大"等模糊性词汇
├── 市场把"叙事上不可能"直接打成"规则上不可能"
└── 案例：MicroStrategy sells any Bitcoin（1.1% Yes，但规则定义模糊）

核心公式

期望收益 = 信息优势 × 市场低效程度 - 手续费

四类规则模糊性（Polymarket Alpha框架）

  1. 时间模糊性 — "年底前"、"选举日后"
  2. 定义模糊性 — "重大"、"官方认定"、"首次"
  3. 来源模糊性 — 结算依赖的"官方来源"未明确
  4. 边界模糊性 — 事件发生的边界条件未覆盖

→ 详见 Polymarket-Alpha轮循策略、Polymarket套利策略研究

三、Buy vs Build决策模式

Polymarket案例：自建引擎 vs 现成工具

决策点 选择 理由
盘口数据获取 直接调Gamma API 完全开放，无需Auth，无需Build
套利识别逻辑 自建（纯计算） 竞争优势在算法，不在工具
规则语义解析 LLM（Claude API） Buy：现有能力足够，比自建NLP省10倍时间
SmartMoney监控 基于open-sol-bot Phase 0先手动验证胜率，再决定自建
链上数据 Helius API 专业节点数据，¥49/月划算

Buy vs Build判断框架

Buy（用现成工具）：
✓ 通用能力（数据API/LLM/基础设施）
✓ 非核心竞争力
✓ 市场上有成熟解决方案

Build（自建）：
✓ 核心竞争力所在
✓ 现成工具不存在或成本过高
✓ 规模化后单位成本会显著下降

教训：SmartMoney跟单是红海（GitHub 230+开源项目），先Phase 0手动验证胜率>45%再投入开发。 → 详见 SmartMoney-Fiona监控系统 调研结论

四、风控共性框架

止损SOP（跨所有项目共识）

进入任何仓位前必须定义：
1. 止损线（价格/事件触发）
2. 加仓条件（具体可量化的催化剂）
3. 止盈目标（避免贪婪）
4. 仓位上限（占组合比例）

各资产类型止损参考

资产 止损设置 来源
美股标的 ATR法（1-1.5×ATR） 美股研究笔记 德鲁肯米勒
A股茅台 MA20下方-4%（¥1385） 投资综合讨论精华
Solana小盘 入场价-25%~30%机械止损 SmartMoney-Fiona监控系统
$CARDS $0.028硬止损 + 链上流动性监控 Cards-CryptoCollect项目

时效性判断（Polymarket专项）

越临近结算 → 规则模糊性价值越高
时间模糊性盘口 → 优先选3月内结束的
流动性检查：大盘口 ≠ 好流动性（volume高但liquidity低=无法执行）

仓位管理原则

高风险资产（Solana Meme/小市值）：组合2-3%，不超5%
中风险资产（港股/ETF）：3-8%，分批建仓
低风险资产（茅台/确定性高）：最高15%，分批买入
预测市场（Polymarket）：规则理解差策略，单笔控制

五、两阶段投研SOP

来自 投资综合讨论精华 stock-research Skill：

阶段一：Ajinomoto Filter快速筛选
├── 直接排除：纯平台/流量公司、市值>$500亿、无营收故事期
├── 核心条件：下游"不得不买"、壁垒来自长期积累、需求从可选→刚需
└── 输出：🟢强买入 / 🟡关注 / 🟠等待 / 🔴回避

阶段二：14维度深度审计
技术护城河 | 商业模式 | 收入增长 | 盈利能力 | 现金流
市场地位 | 管理团队 | 股东结构 | 估值 | 政策监管
产品迭代 | 客户基础 | 供应链 | 分析师共识
