一、投资体系与Skill建设

investment-masters Skill

  • 由查理·芒格、霍华德·马克斯、段永平、斯坦利·德鲁肯米勒、詹姆斯·西蒙斯五位大师数字分身组成
  • 对任意标的进行多维度独立分析，输出委员会裁决报告
  • 已从VPS复制到Fiona节点：~/.openclaw/workspace/skills/investment-masters/

stock-research Skill（两阶段投研系统）

  • 一阶段：Ajinomoto Filter快速筛选
  • 候选公司必须满足：下游"不得不买"、壁垒来自长期积累、市场未充分定价、需求处于从可选→刚需的物理拐点
  • 直接排除：纯平台/流量公司、市值>$500亿、无营收的故事期公司
  • 输出买入评级：🟢强买入 / 🟡关注买入 / 🟠等待 / 🔴回避
  • 二阶段：14维度深度审计（技术护城河、商业模式、收入增长、盈利能力、现金流、市场地位、管理团队、股东结构、估值合理性、政策监管、产品迭代、客户基础、供应链自主、分析师共识）
  • 港股专用版（phase1-hk-agent-prompt.md）：区分直接受益/映射受益/情绪龙头，新增港股通/配售/减持/流动性/做空审计

第四范式（06682.HK）深度研究

  • 综合评分：7.2/10（Alpha，但不是无脑Alpha）
  • 投资建议：合理分批，等回调买，不追高
  • 核心逻辑：港股里少数"AI Agent已落到企业收入里"的票，企业付费验证更硬
  • 风险：港股资金面情绪化，估值容易先跑；无物理门控级别绝对垄断
  • 熔断条件：收入增速降至20%以下/亏损再次扩大/大客户续费不增/股价被情绪打到远高于基本面

金山云（03896.HK）

  • 逻辑：智算云增长+小米/金山生态绑定+首次盈利是关键拐点
  • 风险：客户集中度风险，近期涨幅大

五、操作规范与教训

投研SOP

  1. 新项目必须完成竞品调研+技术路径对比+Buy vs Build
  2. 两阶段流程：GLM快筛→深度审计→CEO裁决
  3. 研报必须在线部署（不用问，每次自动部署）
  4. 所有输出默认使用中文

工具规范

  • 研报HTML部署：Serveo临时隧道（关闭终端后失效），推荐永久托管
  • PPT生成：使用ppt-nano Skill或python-pptx
  • 研报格式：完整HTML交互网页 + 摘要Discord发送
