# AI 人生电影 — 完整市场调研报告

> 调研日期：2026-04-06
> 定位：让每个人都能拍自己（或亲人）的人生电影
> 起源：sz 母亲（69岁）体验 AI 视频后，自发想为已故母亲做纪念视频，并表达付费意愿

---

## 1. 是什么（What）

### 产品定义

**一句话**：输入记忆碎片（照片 + 文字 + 语音），AI 生成完整叙事纪录片/人生电影。

**核心场景**：
- 纪念逝者（清明、忌日）
- 父母生日 / 金婚 / 退休致敬
- 孩子成长回顾
- 婚礼回忆
- 毕业致敬

**产品形态**：
- 输入：5-20 张老照片 + 文字回忆（可由家族群收集）+ 可选语音口述
- 输出：3-10 分钟 AI 叙事短片，含旁白、配乐、AI 生成的过渡画面
- 交付：可下载高清视频 + 可分享链接（类似现有 skill 生成的项目网站）

### 与现有 AI 视频工具的本质区别

| 维度 | 通用 AI 视频工具 | AI 人生电影 |
|------|----------------|-------------|
| 输入 | 文字描述/单张图片 | 多张真实照片 + 文字记忆 + 语音 |
| 目的 | 创意表达/商业内容 | 情感纪念/人生叙事 |
| 用户 | 创作者/营销人员 | 普通人（甚至老年人） |
| 价值 | 效率工具 | 情感连接 |
| 留存动力 | 功能需求（可替代） | 情感需求（不可替代） |

## 2. 为什么（Why）

### 2.1 需求端分析

**真实需求信号**：
- sz 母亲（69 岁，不懂技术）在**未经任何引导**的情况下，自发提出需求并表达付费意愿
- 这是所有项目中唯一出现的"未经引导的自发需求"
- 她已主动拉家族微信群，让亲戚回忆外婆往事 → 自发启动"素材收集"环节

**市场规模**：
- 中国每年死亡人数 ~1000 万，每人平均 5-10 个至亲 → 潜在需求 5000-10000 万人次/年
- 传统纪念视频市场：$500-$25,000/条（美国），¥2000-¥50,000/条（中国）
- AI 版本定价 ¥99-299 → 对传统市场的 10-100 倍降维打击

**情感刚需特征**：
- 不可替代性：这些记忆独一无二，没有"竞品"可以替代
- 紧迫感：亲人在世时间有限，"现在不做以后可能做不了"
- 社交货币：家族群分享天然带动传播
- 节日触发：清明、母亲节、父亲节、重阳节 → 周期性需求

### 2.2 供给端分析

**现有竞品扫描**：

| 竞品 | 定位 | 定价 | 局限 |
|------|------|------|------|
| **StoryShop.ai** | AI 人生故事视频 | $69-99/条 | 依赖视频采访，不适合纪念逝者；$3M seed |
| **Mootion** | 通用 AI 视频 | 免费增值 | 加了纪念 SEO 页面但不是核心场景 |
| **FlexClip** | 在线视频编辑器 | $10-30/月 | 模板化，无 AI 叙事能力 |
| **Picto.Video** | AI 照片转视频 | 按次收费 | 只做动效，不做叙事 |
| **MyHeritage Deep Nostalgia** | 老照片动态化 | $14/月 | 只让照片"动起来"，无叙事 |
| **传统纪念视频公司** | 人工制作 | ¥2000-50000 | 贵、慢、难以规模化 |

**关键发现：市场空白**
- 没有一家公司专门做"AI + 人生叙事 + 纪念"的垂直品牌
- StoryShop.ai 最接近，但要求用户做视频采访，无法用于纪念已逝亲人
- MyHeritage 证明了"让逝者照片动起来"的巨大需求（48 小时 100 万次），但没有做叙事

### 2.3 病毒传播已被验证

**MyHeritage Deep Nostalgia 数据**：
- 2021 年 2 月上线
- 48 小时：100 万次使用
- 18 天：5000 万次使用
- 累计：1.12 亿+ 次
- 22 个国家 App Store #1
- 核心传播路径：家族社交群

**启示**：
- "让逝者照片动起来"已经有 1 亿级需求验证
- "让逝者照片变成完整人生电影"是自然升级
- 传播路径完全一样：家族微信群/WhatsApp/iMessage

### 2.4 核心风险

**Risk 1：Lensa 效应（新奇感消退）**
- Lensa AI 2022 年爆火后留存崩塌
- 原因：功能性工具，用一次就够了
- AI 人生电影的对策：多场景扩展（不止纪念，还有生日/婚礼/毕业），节日周期性触发

**Risk 2：紧迫感缺失**
- "为外婆做纪念视频"很有意义，但"现在就做"的紧迫感不够强
- 对策：绑定节日营销（清明节、母亲节、父亲节、重阳节），限时优惠

**Risk 3：AI 生成质量**
- 当前 AI 视频生成（即梦、Kling）在人物一致性上仍有挑战
- 对策：Phase 0 用真实照片 + AI 过渡画面混合方案，不追求全 AI 生成

**Risk 4：情感产品的信任门槛**
- 用户把珍贵照片交给 AI 处理需要信任
- 对策：先从家族圈（高信任环境）开始，用口碑而非广告获客

## 3. 怎么做（How）

### 3.1 技术可行性

**已有基建**：
- AI 视频全流程 skill 已开发完成（创意 → 剧本 → 出图 → 分镜 → 网站）
- 项目网站生成能力（studio.clawlabs.top）
- Obsidian + generate.py 知识管理管线

**需要新增**：
- 照片上传 + OCR/描述提取
- 基于记忆碎片的叙事脚本生成（Prompt Engineering）
- 照片风格统一化处理（统一色调/滤镜）
- 旁白语音生成（TTS，如 MiniMax / ElevenLabs）
- 最终视频合成（FFmpeg 或 AI 视频拼接）

### 3.2 Phase 0 验证方案（2 周）

**目标**：验证"有人愿意付费"，不需要任何系统建设。

**步骤**：
1. **帮母亲完成外婆纪念视频**（Day 1-3）
   - 素材：家族群收集的照片 + 文字回忆
   - 工具：现有 skill + 手动调整
   - 交付：3-5 分钟纪念短片 + 项目网站
   
2. **观察自然传播**（Day 3-7）
   - 母亲分享到家族群
   - 指标：≥3 人主动询问"怎么做"
   
3. **服务 3 个亲戚，收费 ¥99**（Day 7-14）
   - 全手动服务（不建系统）
   - 指标：≥2 人愿意付费
   
4. **观察二次传播**（Day 14+）
   - 他们是否分享到自己的朋友圈/家族群
   - 是否有陌生人通过他们找过来

**Phase 0 通过标准**：
- ≥2/3 亲戚付费
- 出现至少 1 次二次传播
- 全部通过 → 升级为主线 A，短剧工作室降为 A2

### 3.3 MVP 方案（Phase 0 通过后，4 周）

**只在 Phase 0 验证通过后才建设**：
- 简单的微信小程序或 H5 页面
- 上传照片 + 填写文字回忆 → 生成视频
- 定价：¥99（基础版，3 分钟）/ ¥199（标准版，5 分钟）/ ¥299（精致版，10 分钟+定制旁白）
- 支付：微信支付

### 3.4 增长策略

**第一阶段：家族裂变**
- 每个满意用户分享到家族群 → K-factor 天然 > 1
- 每个家族群 10-50 人，每人都有自己的家族群 → 指数级传播

**第二阶段：节日营销**
- 清明节（4 月）：纪念逝者主题
- 母亲节（5 月）：为妈妈做人生电影
- 父亲节（6 月）：为爸爸做人生电影
- 重阳节（10 月）：敬老主题

**第三阶段：内容营销**
- 用户授权后，精选作品发布到短视频平台
- "AI 帮我拍了外婆的人生电影"→ 情感类内容天然高传播

4. **家族群已激活**：母亲已自发组织亲戚回忆，素材收集已开始

**时间线**：

## 5. 竞争壁垒分析

### 5.1 短期壁垒（6 个月内）

- **先发品牌**：第一个"AI 人生电影"品牌认知
- **家族网络效应**：每个用户带来整个家族群
- **情感口碑**：这类产品的口碑传播力远超工具类产品

### 5.2 长期壁垒（12 个月+）

- **用户生成内容飞轮**：更多作品 → 更多分享 → 更多用户 → 更多作品
- **叙事模板库**：积累各种人生阶段的叙事模板（战争年代、下乡、改革开放...）
- **情感数据壁垒**：理解中国家庭的情感表达方式

### 5.3 可能的竞争威胁

- **大厂进入**：字节/腾讯可能做类似功能，但大厂很难做情感类垂直产品（调性问题）
- **通用 AI 工具升级**：Sora/即梦等可能加入"照片变视频"功能，但叙事+情感+仪式感是差异化
- **传统影视公司转型**：有内容能力但缺 AI 效率

## 6. 定价策略

### 6.1 参考定价

| 方案 | 价格 | 内容 | 目标用户 |
|------|------|------|----------|
| 基础版 | ¥99 | 3 分钟短片，AI 旁白 | 尝鲜/验证需求 |
| 标准版 | ¥199 | 5 分钟，定制旁白+配乐 | 主力产品 |
| 精致版 | ¥299 | 10 分钟，多章节叙事 | 重度用户 |
| 定制版 | ¥499-999 | 人工介入调整，多轮修改 | 高端/特殊场景 |

### 6.2 成本结构（per 单）

- AI 视频生成：¥5-15（即梦/Kling 按秒计费）
- AI 旁白生成：¥1-3
- 服务器/存储：¥1-2
- **边际成本：¥10-20/单**
- **毛利率：80-90%**

### 6.3 定价心理

- 传统纪念视频 ¥2000-50000 → ¥99-299 是"不可思议的便宜"
- 一束花 ¥100-300 → 同价位但"更有意义"
- 情感产品的价格弹性低 → 用户不会因为贵 ¥50 而放弃

## 7. 为什么可能比短剧工作室更优

| 维度 | AI 人生电影 | AI 短剧工作室 |
|------|-------------|---------------|
| 需求验证 | 有（母亲自发需求） | 无（假设需求） |
| 变现模式 | 用户直接付费 | 流量间接变现 |
| 传播方式 | 天然病毒传播（家族群） | 需要投放 |
| 利润分配 | 100% 自己的 | 55:45 分成 |
| 技术杠杆 | 相同管线 | 相同管线 |
| 竞争格局 | 空白市场 | 红海（大量 AI 短剧工作室） |
| 用户粘性 | 高（情感连接） | 低（内容消费） |
| 天花板 | 全球华人家庭 | 依赖平台算法 |

**结论**：如果 Phase 0 验证通过，AI 人生电影应升级为主线 A。

- **日本**：祖先崇拜文化，お盆（盂兰盆节）是强触发场景
- **韩国**：祭祀文化，有类似需求
- **全球**：母亲节、父亲节是全球性节日
