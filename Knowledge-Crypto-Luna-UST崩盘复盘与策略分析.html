Luna/UST 崩盘复盘与策略分析

[!abstract] 核心教训 人赚不到认知之外的钱。看到价格下跌的表象，没有明白背后的机制原理，是错过千万级机会的根本原因。

投资复盘 #认知升级 #crypto #luna #稳定币

一、新Luna策略分析 (2022-05-18)

Do Kwon提案：Fork发行新Luna

分配规则（10亿新Luna）： - 25%（2.5亿）→ Community Pool（加池子），LFG资产余额约2.5亿，估计初始价格$1 - 1% → 开发人员紧急使用 - 4% → 开发（1年Cliff, 4年Vesting） - 35% → 5月7日攻击前快照的Luna Holder

关键信息： - 5.27 04:00 UTC快照，新链没有UST - 当时UST总量约112亿，空投2.5亿，1 UST = 0.022 新Luna，TGE 10%

风险分析

新方案得到部分生态项目方支持，但受损严重的知名项目（Anchor、Mirror）无一表态支持。方案完全没有考虑VC及交易所利益，且CZ明确表示反对——新币能否上所存疑。

五种交易策略

策略 逻辑 风险
1. 买LUNC等空投 1亿空投给6.5万亿Holder，LUNC低于0.0000015才稳赚 价格已反映预期
2. 买UST等空投 1 UST = 0.0022新Luna（算10% TGE） UST可能继续脱锚
3. 做空LUNC 空投后变Meme，有70%~700%下跌空间 市场不理性可能爆拉
4. 做空UST 保底价0.022，有~75%下跌空间 同上
5. 开盘冲新Luna ⭐ 最多3%抛压，好拉盘；开盘$1冲 盘子0.25b，新资金是否足够？

结论：做空旧币最大风险是Meme式非理性拉盘。开盘冲新Luna综合亏损可能性不大。

二、五月复盘 (2022-05-20)

错过了什么

5.7至5.12，Luna史诗级崩盘1000万倍，到底部又反弹上千倍。在希尔顿酒店有大把时间，结果只是搬BUSD/USDC的砖。

最佳策略复盘

核心认知：看价格只是表象，要看机制

  1. 刷推特获取一手信息 → 在UST锚定机制未关闭时一直开空（机制不关闭就无限下跌）
  2. 机制关闭时抄底 → 跌了千万倍，涨几百上千倍很正常
  3. 链上搬砖 → 1 UST = 1美元的链上兑换套利
  4. 借贷套利（Venus等）→ 超出当时能力圈

为什么没抓住？

  • 没有深度学习UST锚定机制 → 认知不到位
  • 没有亲身体验Terra上UST&LUNA的锚定流程 → 没有入局
  • 看了某篇文章说链上兑换门槛高就放弃了 → 吃别人嚼剩的信息

"非常可惜，如果这次能赚几千万，也不用再苟且于蚂蚁的这份工作。" "这次算是学了一堂价值千万的课。我现在最多就是A7的认知，希望这堂课能把认知提升到A8。"

四条行动准则

  1. 定期复盘 — 最低每周一次，前期每天一次，先坚持一个月养成习惯
  2. 亲身入局 — 一定要完整体验流程（如稳定币提现美元，危机时心里才有底）
  3. 掌握源头信息 — 自己分析事实vs推测，刷项目方推特获取一手信息
  4. 保持理智 — 不FOMO不恐慌，掌握前三点后自然能做到

三、四月复盘与反思 (2022-04-24)

整个4月错过了很多机会，只抓住了一个。

五个根因

  1. 没有经常复盘 — 没有复盘就没有反思，没有沉淀，错了就阿Q精神安慰自己
  2. 心态太急躁 — 总想一夜暴富，觉得思考是浪费时间，把事情想得都很简单
  3. 行动力不强 — 觉得好的又不能马上执行
  4. 没有聚焦 — 太散了，看到别人赚钱就想去，没有自己的思考和风格
  5. 性格问题 — 短期利益看得太重，表现太直接

只有学习 + 行动 + 复盘，不断总结，不断反思，形成闭环，才可能提高。

四、读书笔记《韭菜的自我修养》(2023-08-03)

核心收获

  1. 教是最好的学，写是最好的读 — 能写出来、能教别人才是真正理解了
  2. 一次写作，终身传播 — 一份时间复制成无数份，提高效率、延长生命
  3. 牛市终结的根本原因：入场资金枯竭 — 加息→大资金存美国吃无风险利息
  4. 痛苦 + 反思 = 成长 — "我不缺痛苦，但极缺反思，所以一直没有办法持续成长"

认知进化轨迹

时间 阶段 关键认知
2022.04 散乱期 发现问题：不复盘、急躁、不聚焦
2022.05 觉醒期 Luna教训：要看机制不看价格，要入局不做旁观者
2023.08 沉淀期 痛苦+反思=成长，写作是最好的学习
2026.03 系统期 用AI+自动化构建知识库，系统性沉淀认知

从2022年"价值千万的课"到2026年建立AI驱动的自动化知识库，这条路走了4年。现在的Obsidian+OpenClaw体系，本质上就是在践行当初总结的四条准则：定期复盘（自动化）、深入本质（AI辅助分析）、源头信息（多平台抓取）、保持理智（数据驱动决策）。
