LibTV AI 视频生成平台

[来源: workspace/skills/libtv-skill / Claude Code 2026-03-28]

平台定位

LibTV 是 LiblibAI 推出的 AI 视频创作平台，同时为人类创作者和 Agent 设计，已在 OpenClaw 安装 libtv-skill。

核心能力矩阵

能力 说明
文生图 多种顶级模型
文生视频 Kling 3.0/O3、Seedance 2.0 等
图生视频 参考图生成动态视频
视频续写 延长已有视频
局部编辑 元素替换、镜头调整、风格迁移
短剧生成 一句话 → 剧本 → 分镜 → 成片
MV 生成 音乐 + 风格 → MV
产品广告 TVC/宣传片复刻与生成

支持模型

  • Seedance 2.0
  • Kling 3.0 / O3
  • Wan 2.6
  • NanoBanana（图像）
  • Midjourney
  • Seedream 5.0

Agent 调用方式（libtv-skill）

所有操作通过 自然语言消息 发送到会话，Agent 自主编排工作流：

1. 创建会话 / 发消息 — "生一个动漫视频"
2. 查询进展 — 轮询 sessionId 获取结果（复杂任务耗时较长）
3. 切换项目 — 绑定不同 projectUuid
4. 上传文件 — 编辑已有图片/视频时先上传到 OSS
5. 下载结果 — 批量下载生成的图片/视频

配置要求

  • 环境变量：LIBTV_ACCESS_KEY
  • 依赖：python3

触发场景

  • 用户说：画只猫、做个海报、视频换风格、帮我复刻这段视频
  • 涉及：liblib、libtv、上传参考图/视频
  • 任何 AI 图片/视频的创作、生成、编辑、修改需求
