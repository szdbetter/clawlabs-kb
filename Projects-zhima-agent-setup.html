芝麻 Agent 初始化记录

日期: 2026-03-16
频道: Discord #family › Reese
归档时间: 2026-03-19

背景

岁月为老婆 Reese（小蓝/蓝校长/哼哼怪）创建了专属 AI Agent，配置 ID 为 zhima，对应飞书机器人名为"芝麻"。

问题诊断

问题: zhima agent 没有独立 workspace
根因: OpenClaw 不会自动创建配置中指定的 workspace 目录，/root/.openclaw/workspace-zhima 从未手动创建，导致芝麻无法初始化。

解决步骤

  1. 手动创建 /root/.openclaw/workspace-zhima 目录 ✅
  2. 生成芝麻专属配置文件 ✅
  3. SOUL.md — 芝麻人设、性格、禁忌
  4. IDENTITY.md — 身份卡片
  5. USER.md — 女主人档案
  6. 飞书密钥由岁月自行配置（FEISHU_APP_ID_ZHIMA / FEISHU_APP_SECRET_ZHIMA）✅

芝麻人设概要

  • 名字: 芝麻（飞书机器人）
  • 女主人: Reese（小蓝 / 蓝校长 / 哼哼怪）
  • 背景: 英国曼彻斯特大学研究生毕业，从事教育管理工作
  • 风格: 温柔俏皮有脑子，不油腻不废话
  • 核心能力: 教育管理支持 + 日常陪伴 + 生活助手 + 英文资料处理
  • 技术问题: 推回给岁月处理

技术配置

  • 模型: anthropic/claude-sonnet-4-6
  • workspace: /root/.openclaw/workspace-zhima
  • 飞书: 独立账号绑定（zhima 条目）
  • 与 Reese 主 Agent 关系: 完全独立，不共享 workspace 和人设
