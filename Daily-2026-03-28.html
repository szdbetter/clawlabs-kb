💡 今日核心洞察

洞察1：弱验证归零 = 理论正确，但工程缺口明确

弱验证8周0交易不是策略死了，而是「标签粒度不足」这一工程缺口的确认。策略逻辑（Elon推文数可预测 → 市场定价存在系统性偏差）仍未被证伪，只是需要更细粒度数据才能验证。下一步卡点在X API付费权限，这是可解决的工程问题。

洞察2：企业AI顾问是比纯To C更有杠杆的收入路径

一单企业咨询合同（政府方向）能覆盖数月运营成本，且建立的信任关系可持续深耕。老板的背景（区块链To G经验 + 讲师经验）在这条路上有显著竞争壁垒。

洞察3：Claude Code on Mac订阅账号 = 403坑

Mac端Claude Code用订阅账号会遇到403 forbidden，根本原因是订阅账号无API Key，而Claude Code默认走API。解决方案：用 claude auth login 走OAuth而非API Key模式。

洞察4：AI生成文件分发 = 临时OSS链接是陷阱

MiniMax / 其他AI平台生成的OSS链接普遍24h内过期，长期分发必须上传到自控仓库（GitHub/CDN）。"GitHub gh-pages + raw.githubusercontent.com"是零成本永久链接方案。
