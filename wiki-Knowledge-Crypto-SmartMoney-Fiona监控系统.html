SmartMoney-Fiona监控系统

来源：Discord #SmartMone-Fiona (thread_id: 1482721690751467694, 420条) 提取时间：2026-03-20

一、项目定位

核心赚钱逻辑： 1. 从GMGN获取热门Meme币的盈利地址进行统计 2. 把高盈利地址抓取入库，形成自己的SmartMoney地址库 3. 监控这些地址的买币情况，作为跟单信号

三阶段目标： 1. Phase 1：先做监控，通过监控获取信息信号并发送Discord提醒（自用） 2. Phase 2：实现跟单 3. Phase 3：未来是否做付费群另行决定

二、调研结论（2026-03-19 Opus调研报告）

竞品全景

类型 产品 价格 特点
一体化平台 GMGN 免费 Meme专精，Solana最强
一体化平台 Cielo $59-199/月 多链
TG Bot BullX/Trojan/Photon 0.9-1%手续费 免费订阅
链上分析 Nansen/Arkham $150+/月 分析为主无跟单
开源项目 ChainBuff/open-sol-bot 免费 404⭐ Python，Apache-2.0

赛道判断

  • 红海：GitHub 230+ 个开源项目，成熟商业工具全覆盖
  • SM跟单实际胜率：社区反馈约30-40%（Meme币90%归零）
  • GMGN爬取：不稳定，Cloudflare严防，随时可能失效

建议路径

阶段 动作 成本 条件
Phase 0 手动用GMGN+BullX跟单2周，记录胜率 $0 立即
Phase 1 基于open-sol-bot搭MVP + Helius Starter + Discord信号 $49/月 Phase 0胜率>45%
Phase 2 Helius gRPC实时 + AI SM评分引擎 $999/月 Phase 1 ROI为正

三、技术架构

数据采集层

  • 采集主体：Fiona节点（macOS，住宅IP，永远在线）
  • 方式：Playwright headed模式（绕过Cloudflare）
  • 频率：1h/次（全部统一为1小时）
  • 关键API： ``` # 热门代币列表 GET /defi/quotation/v1/rank/sol/swaps/1h

# 代币持有者+盈亏 GET /vas/api/v1/token_holders/sol/{token}

# 钱包交易记录 GET /vas/api/v1/wallet_activity/sol?wallet={addr}&token={token} ``` - ⚠️ 重要：GMGN必须在页面上下文中调用API，需Playwright带cookie访问，不能直接curl

数据传输

  • Fiona采集 → HTTP同步 → VPS存储

存储层（VPS SQLite）

tokens          -- 热门代币列表
wallets         -- SM钱包地址库
wallet_token_profits  -- 每个钱包在各代币上的盈亏
signals         -- 跟单信号推送记录

展示层

  • 本地Dashboard：http://127.0.0.1:1980

四、SM评分体系

入库标准

  • 跨3+代币盈利
  • 非bot
  • 累计盈利 > $10K

评分维度（权重）

  1. 跨币盈利数（30%）
  2. 累计利润（25%）
  3. 胜率（20%）
  4. 活跃度（15%）
  5. 标签信誉（10%）

五、首批数据验证（2026-03-19，Fiona实测）

100代币规模扫描结果

  • 扫描代币：69个（API实际返回）
  • 唯一盈利钱包：1,851个
  • 出现在2+代币：129个
  • 出现在3+代币（真SM）：23个

规律分析

  1. 真正跨币盈利SM极少：69币中仅23个钱包在3+币上盈利
  2. 标签是金矿：axiom、photon等是交易工具标签，标记专业交易者
  3. fresh_wallet大量出现：新钱包赚大钱 = 可能是内部人/狙击手
  4. 需扩大样本：100+代币×多个时间段才能筛出稳定盈利地址
