AI视频提示词工程与分镜方法论

来源

docs/AI视频/创作/短剧写作教程/20260417/

一、提示词核心公式

主体 + 动作 + 场景 + 镜头 + 光线 + 质感 + 氛围 + 限制

五大量化等级

  • 等级1（说明句）：一个女生在街上走
  • 等级2（画面句）：一个穿红裙子的女人在夜晚街头行走，路灯昏黄
  • 等级3（镜头句）：中景从背后轻微手持跟拍，潮湿地面反射灯光
  • 等级4（电影句）：低饱和，胶片颗粒，压抑而孤独的电影氛围
  • 等级5（专业导演句）：完整约束 + 负面限制词

二、提示词分层结构

层级 内容 示例
层1 主体 谁？年龄/性别/穿着/状态
层2 动作 在干嘛？可视化动词
层3 空间 在哪儿？地点+时间+天气+空间特征
层4 镜头 怎么拍？景别+运镜+机位
层5 光线 光源位置+颜色+强度+效果
层6 质感 胶片/数字/写实/动漫/梦境
层7 情绪 少而准，如"孤独而安静"
层8 限制 不要什么？畸形手/水印/卡通风

三、分镜写法核心原则

  1. 一镜一重点：一条提示词只解决一个主要任务
  2. 镜头顺序=观众理解顺序：先知道什么后知道什么
  3. 镜头变化要有层次：空间/行动/关系/情绪/细节/转场
  4. 每个镜头都要可生成：AI最怕复杂动作、多角色、长时间变化

四、各平台差异化写法

平台 风格 重点
即梦 中文电影感连贯描述 氛围、光线、完整画面
可灵 动作+镜头调度明确 动作、运镜、空间变化
Runway 英文简洁结构 主体、动作、镜头、风格
Pika 短句视觉导向 单镜头、直观动作

五、专业提示词检查表

  • [ ] 主体是谁，够清楚吗？
  • [ ] 动作是可视化的吗？
  • [ ] 场景有空间信息吗？
  • [ ] 镜头语言明确吗？
  • [ ] 光线来源清楚吗？
  • [ ] 质感词具体吗？
  • [ ] 情绪词是否过多？
  • [ ] 有没有负面限制？
  • [ ] 有没有相互矛盾？
  • [ ] 这条更像导演指令而不是散文？

六、Seedance 2.0 万能公式（2026更新版）

@参考 + 主体 + 动作细节 + 场景环境 + 光影色彩 + 镜头语言 + 画面风格 + 画质标准 + 稳定约束 + 画幅时长 + 负面提示词

2026更新版权重优先级

多模态参考 ＞ 动作细节 ＞ 稳定约束 ＞ 镜头语言 ＞ 画质风格

动作核心技巧

  • 不写笼统动作，写连续、有起止、慢节奏、有细节的完整动作
  • 优先修饰词：缓慢、轻柔、连贯、自然、流畅、平稳、不僵硬、匀速、无顿挫

运镜稳定要求

手持稳定、无抖动、丝滑流畅、无卡顿、画面匀速运动、无忽快忽慢

分镜写法

按时间顺序用【镜头X 时长Xs】标记，单条视频镜头不超过4个

七、导演分镜交材体系（专业版）

必须包含的16个模块

  1. 项目概述 2. 导演总阐述 3. 视觉风格总说明 4. 场次总览
  2. 单场详细交材 6. 分镜镜头清单 7. 摄影专项 8. 灯光专项
  3. 美术专项 10. 声音专项 11. VFX/特殊执行 12. 剪辑预想
  4. 音乐预想 14. 风险与备选 15. 版本管理 16. 附录

镜头清单标准字段

镜号/场次/景别/机位/运镜/焦段/时长/画面内容/动作/台词/声音/转场/道具/特效/优先级/备注

优先级体系

  • A类：必拍镜头（剧情核心，不拍就没法剪）
  • B类：安全镜头（保证剪辑成立）
  • C类：可删镜头（时间不够时可牺牲）
  • D类：补充镜头（氛围/细节/转场）

六大原则

  1. 戏剧先于镜头
  2. 空间先于剪辑
  3. 镜头不是记录动作，而是组织注意力
  4. 每场戏都要有"镜头策略"
  5. 兼顾艺术表达和执行落地
  6. 导演交材是"给全组看"的
